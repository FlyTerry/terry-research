主焦煤价差分析（2026年1-4月）


据Mysteel分析，2026年1-4月炼焦煤价格呈现以下特征：


1. 整体趋势：先抑后扬，各煤种整体性上涨，价格中枢上移

2. 价差收窄：高硫主焦煤涨幅领先（+31%），低硫（+17%），主焦煤品种价差逐渐收窄

3. 驱动因素：

- 配煤技术突破，高硫主焦煤配比上限增加

- 脱硫能力提升，高硫煤使用更稳定

- 美国进口煤政策变化，增加国内高硫采购

- 矿山库存维持低位，报价坚挺

- 华北华东焦企盈利较好，采购灵活性高


高硫主焦煤涨幅领先（+31%）的深层逻辑：

（1）价格基数效应：高硫主焦煤（S≤2.0%）2025年底价格基数低（约600-700元/吨），相同绝对涨幅下，百分比涨幅更大。低硫主焦煤（S≤0.6%）基数较高（约1,450元/吨），涨幅相对温和。

（2）供应收缩更严重：2025年山西停产煤矿中，高硫矿井占比更高（柳林寨崖底S≤2.2%、孝义兑镇S≤3.0%等）。低硫优质矿井（如安泽、离柳）受影响相对较小。供应收缩 → 价格弹性更大。

（3）下游降本驱动：钢厂利润承压（盈利率63.2%），主动优化配煤成本。高硫主焦煤性价比突出（价格低、热强度足够），配比从15-20%提升至25-30%。需求增加 → 价格上涨。

（4）进口替代有限：美国高硫主焦煤进口政策收紧，国内高硫煤替代需求增加。而低硫主焦煤的进口替代来源更多（蒙古、澳洲），价格波动相对平缓。

（5）价格传导滞后：低硫主焦煤价格受长协合同约束，调整频率低。高硫主焦煤以现货交易为主，价格对市场变化的反应更灵敏、涨幅更猛。


数据验证：据MySteel数据，2026年Q1高硫主焦煤从650元/吨涨至850元/吨（+31%），同期低硫主焦煤从1,450元/吨涨至1,700元/吨（+17%）。价差从800元/吨收窄至850元/吨，反映配煤成本差异的收敛。

六、主焦煤价格与钢厂铁水产量关联性分析


价格与下游需求关联分析


主焦煤作为钢铁生产的重要原料，其价格走势与钢厂生产活动（以铁水产量为代表）存在密切关联。本节基于我的钢铁网（Mysteel）公开数据，通过叠加分析2024-2026年主焦煤价格与钢厂日均铁水产量数据，揭示两者之间的关联关系。


数据走势对比（2024年1月 - 2026年5月，Mysteel数据）


主焦煤价格走势：

  • 2024年初：约2,500元/吨（高位）
  • 2024年全年：持续下跌至1,450元/吨，跌幅约42%
  • 2025年Q1：维持低位震荡，1,450-1,480元/吨
  • 2025年H1：Mysteel国产炼焦煤指数从1,273元/吨降至936.5元/吨（6月，近8年低点）
  • 2025年H2：触底反弹，11月中旬回升至1,407元/吨
  • 2026年Q1：先抑后扬，高硫主焦煤(+31%)涨幅领先
  • 2026年4月：低硫主焦煤1,525元/吨，同比涨17%
  • 2026年5月：唐山低硫主焦煤1,595元/吨，邯郸1,490元/吨
  • 2026年6月：安泽低硫主焦煤1,800元/吨，5月单月涨140元/吨

钢厂日均铁水产量（Mysteel数据）：

  • 2024年全年：维持在229-234万吨/日区间
  • 2025年上半年：高位运行，峰值达234万吨/日
  • 2025年下半年：略有回落，维持在229-232万吨/日
  • 2026年Q1：回升至238万吨以上
  • 2026年5月最新：240.81万吨/日（环比+1.48万吨），钢厂盈利率63.2%

关联性分析


1. 价格与产量的负相关关系（2024年）


2024年主焦煤价格大幅下跌期间（2,500→1,450元/吨），铁水产量保持相对高位（229-234万吨/日），反映：

  • 钢厂生产积极性较强，但利润承压导致对原料价格敏感度提升
  • 供应宽松格局下，价格缺乏上行支撑
  • 下游需求（房地产、基建）偏弱，传导至原料端

2. 2025年下半年的正向联动


2025年7-11月，Mysteel国产炼焦煤指数从936.5元/吨反弹至1,407元/吨（涨幅50.3%），同期铁水产量虽略有下降但仍在229万吨/日以上，显示：

  • 钢厂利润边际改善，原料采购意愿增强
  • 供给端约束（山西矿难、安监升级）推动价格上行
  • 下游需求预期好转，产业链信心修复

3. 2026年的量价齐升（Mysteel最新数据）


2026年1-5月呈现"量价齐升"新格局：

  • 日均铁水产量突破240万吨，达到240.81万吨（近年高位）
  • 钢厂盈利率63.2%，盈利状况改善
  • 低硫主焦煤从1,525元/吨涨至1,800元/吨（5个月涨幅275元）
  • 高硫主焦煤涨幅领先（+31%），品种价差收窄
  • 主焦煤库存低位运行，矿山报价坚挺

4. 供需博弈的核心矛盾


  • 需求端：铁水产量高位（240万吨+）提供刚性支撑，但中长期下行趋势未变
  • 供给端：优质低硫主焦煤资源稀缺，山西开工率虽达93%但安监约束产量释放
  • 结构变化：高硫主焦煤配比提升、品种价差收窄，反映下游降本趋势
  • 价格弹性：低硫主焦煤等骨架煤种价格弹性更大，6月已回升至1,800元/吨

  • 2024年：主焦煤均价1,753-1,972元/吨，价格高位
  • 2025年：主焦煤均价~950-1,050元/吨，暴跌45-47%

公司业绩总结

  • 2025年：12家公司中，2家亏损（美锦-11.23亿、开滦-0.32亿），10家盈利但利润大幅下滑
  • 2026年：价格回升，预计大部分公司利润大幅反弹

投资逻辑总结

  • 稀缺资源：拥有优质主焦煤资源的公司（平煤股份、首钢资源注：首钢资源实际硫含量1.0-1.5%，已转型中高硫焦煤）
  • 高分红：派息率高的公司（首钢资源97%、潞安环能50%）
  • 高弹性：价格弹性高的公司（平煤股份、蒙古焦煤、淮北矿业）

投资逻辑


  • 供应端：2025年全国原煤产量48.3-48.5亿吨（同比↑1.2%-1.4%），进口炼焦煤约1.2亿吨（同比↓2.66%-9.6%），安全生产、稳定供应政策持续推进
  • 需求端：受钢铁行业下游订单减少及电炉钢占比提升影响，对焦煤需求形成抑制，但2026年价格已显著回升（低硫主焦+94%）
  • 稀缺资源禀赋：拥有优质低硫主焦煤资源的公司具备长期竞争优势
  • 高分红属性：部分公司分红比例较高（如首钢资源派息率约97%、潞安环能50.16%、淮北矿业44.71%）
  • 高弹性：价格弹性高的公司（平煤股份、蒙古焦煤、淮北矿业）在价格回升周期中利润弹性显著
