核心结论摘要

  • 业务定位：中蒙跨境物流龙头，蒙古焦煤进口核心物流服务商，主营"物流+供应链贸易"一体化。非洲为第二增长曲线，复制"陆港节点+集疏运短腿"卡位模式
  • 中蒙业务：占营收约60-65%，核心资产为甘其毛都、策克口岸物流基础设施。甘其毛都日均通车量为景气度先行指标——看趋势不看绝对值，区分供给/需求/效率端不同驱动因素
  • 非洲业务——核心逻辑：嘉友赚的是节点过路费，不是干线运费。陆港毛利率59.48%是运输（26.73%）的2.2倍、贸易（10.47%）的5.7倍。2026Q1已出现结构升级信号：营收-15.78%但净利+25.18%
  • 迪洛洛——纯期权：3.63亿美元投资，219.4km公路+陆港，嘉友51%+紫金49%。但尚未开工，最早2030年建成。当前7份券商研报盈利预测均不含其贡献——市场估值未透支此预期
  • 紫金——上游阀门：持股17.26%+迪洛洛项目联合投资+刚果金铜矿物流独家绑定。2025-2026年卡莫阿减产直接拖累嘉友业绩，验证了"阀门开多大，物流量就多大"的双刃剑属性
  • 四条通道：洛比托（西路/最快但公路未建）、达累斯萨拉姆（东南路/在赚钱但路线长35天）、坦赞铁路（远期期权/修复中）、鲸湾港（南路备用）——嘉友在每条通道的咽喉处卡位设收费站
  • 财务稳健：总资产约80.9亿元，资产负债率仅23.78%，有息负债仅3.56亿元（有息负债率4.28%），经营现金流11.31亿元
  • 敏感性：蒙古焦煤进口量/价格每变动10%，对净利润影响约±8-12%。但随着高毛利陆港占比提升，整体盈利对焦煤价格的敏感性有望降低

1.3 蒙古焦煤业务详解

📋 业务模式：

  • 物流模式：蒙古煤矿→跨境运输（嘉友承运）→中国口岸（甘其毛都/策克）→终端钢厂，嘉友收取全程物流服务费
  • 供应链贸易模式：嘉友以自身名义向蒙古煤矿采购焦煤→运输至中国→销售给国内钢厂/焦化企业，赚取采购与销售价差（含物流服务收入）
  • 上游合作：与蒙古主要焦煤企业（如Energy Resources LLC、Mongolian Mining Corporation等）建立长期股权/合作协议，保障货源稳定
  • 下游客户：国内大型钢厂及焦化企业（如包钢、河钢、鞍钢等）

第三章 蒙古焦煤业务深度分析

3.1 蒙古焦煤行业背景

📋 蒙古焦煤核心地位：

  • 储量全球领先：蒙古焦煤储量约40-50亿吨，主要分布于南戈壁地区（塔本陶勒盖煤矿等）
  • 中国最大进口来源：2025年中国进口蒙古焦煤约4,000万吨，占中国焦煤进口总量约45-50%
  • 运输通道集中：主要通过甘其毛都口岸（约占60%）、策克口岸（约占30%）进入中国
  • 价格高波动：蒙古焦煤价格受中国钢铁需求、中蒙关系、运输成本等多因素影响，年度波幅可达±30%

3.4 甘其毛都口岸日均通车量——景气度先行指标

📋 为什么关注通车量：甘其毛都是中蒙边境最大的焦煤通道，蒙古焦煤通过卡车一辆一辆拉入中国。日均通车量即每天通过该口岸的运煤卡车数量，每辆重卡载重数十吨，通车量×载重≈日过境煤量，是嘉友中蒙业务的景气度先行指标。

3.4.1 通车量与嘉友业绩的三层关系

关系层级机制确定性权重
第一层：正相关通车量涨→物流服务费涨、贸易量涨，正相关关系直接约50%
第二层：前瞻指标通车量领先收入确认（煤从运输到销售到记账有时滞），可作业绩前瞻。但领先时长取决于公司结算节奏，不完全确定约30%
第三层：份额波动全口岸通车量≠嘉友份额。嘉友切多少取决于自身物流合同、客户关系、仓储条件。份额波动削弱通车量与业绩的相关性约20%

3.4.2 通车量的三重驱动因素

驱动因素方向对嘉友影响
蒙古煤供给矿区产能、出口政策、天气路况→影响运到口岸的煤量供给多了可能压煤价（利空贸易毛利、利好物流量）
中国需求钢厂开工率、焦煤需求→影响进口商拉货意愿需求旺推煤价（利好贸易毛利）
口岸通关效率海关查验速度、口岸基建、天气/政策变化→影响日通车上限纯利好：量增服务费涨，不压价

如何运用该指标：

  • 看趋势不看绝对值：同比大涨需关注环比是否走弱——环比下行可能预示景气度见顶
  • 区分驱动因素：通车量上涨时判断是供给端、需求端还是效率端驱动，三者对嘉友利润影响方向不同
  • 甘其毛都不是唯一口岸：策克、满都拉也是中蒙焦煤通道，但甘其毛都是最大、最主要的焦煤通道（约占60%）
  • 配合焦煤价格使用：通车量+焦煤价格双升是最佳组合（量价齐升）；通车量升+价格跌需警惕"增量不增利"

4.2.2.1 关键澄清：嘉友赚的是节点过路费，不是干线运费

洛比托走廊的资产归属需精确区分：

  • 洛比托铁路（1739公里干线）：安哥拉段由LAR联合体运营（托克Trafigura 49.5% / Mota-Engil 49.5% / Vecturis 1%，30年特许权）；刚果金段由刚果金政府与Mota-Engil的PPP合作运营（2026年7月10日获批）。嘉友没有直接参股铁路。
  • 嘉友持有的是219.4公里集疏运短腿 + 迪洛洛陆港节点：角色是矿→嘉友公路→嘉友陆港→洛比托铁路→洛比托港。嘉友卡的是矿区到铁路装车点之间的"最后一公里"和边境陆港节点。

"不可替代的收费站"逻辑：

  • 干线运输（1739公里铁路）别人持有、可替代、价格透明。嘉友不持有，也不应被当成嘉友的价值。
  • 清关、仓储、换装发生在陆港节点——货物要上火车、要出口，必须在边境装车点过海关、进保税仓、完成换装。这是物理上绕不开的节点。谁持有这个节点，谁收刚性过路费。
  • 219.4公里公路单独看可竞争（矿企理论上可自修支线接铁路），但嘉友公路的终点是嘉友自己的陆港，陆港是铁路装车的必经节点。嘉友把"走我的路"和"在我的港装车清关"打包成服务包，降低被绕开概率。

财务数据验证——节点利润厚度远超运输：

业务板块2025年报毛利率利润厚度对比
陆港项目服务59.48%基准
跨境多式联运综合物流26.73%陆港的45%

2026年一季报更直接：营收19.33亿元同比-15.78%，归母净利润3.28亿元同比+25.18%——收入降利润反涨，说明高毛利陆港业务占比在提升，盈利结构在优化。

4.2.2.2 铁路扩容的影响：中性偏多，但有时序前提

三重机制叠加分析：

机制方向影响分析
价格挤压偏空铁路运输成本显著低于公路，LAR扩容后运价走低，矿企会拿铁路价压嘉友公路+陆港报价。公路段运费单价承压，但节点费（清关/仓储）相对刚性，不易被压。
模式替代风险若铁路修出自有支线或矿企自建专用线，能绕过嘉友219.4公里公路。但嘉友公路终点是嘉友陆港（铁路装车必经节点），完全绕开难度大。
总量溢出利多铁路扩容→走廊总铜量大幅提升→更多货必须经过迪洛洛陆港清关装车→节点费和仓储量跟涨。

⚠️ 关键时序问题：LAR CEO表示铁路DRC段预计2028年全线贯通。但嘉友迪洛洛即使2026年开工，最早2030年才建成。如果铁路2028年贯通而陆港2030年才建成，中间这两年货物走洛比托铁路但不经过嘉友陆港——总量溢出机制在初期是空转的。嘉友要吃铁路扩容红利，前提是陆港先建好。

净效应判断：陆港毛利率59.48%远高于运输26.73%，节点利润更厚。如果大部分利润来自节点费而非运费，铁路挤压的影响有限。数据链支持中性偏多：铁路扩容做大走廊总盘子→嘉友节点量受益，仅运费单价承压。前提是陆港先建好。

4.6 非洲商业模式深度剖析：卡位咽喉收"过路费"

核心逻辑

嘉友在非洲做的不是"修一整条路"，而是卡住矿区到港口之间的咽喉节点——口岸、陆港、收费站。刚果金是内陆国，无出海口，矿挖出来必须通过邻国港口运出去。嘉友在这些出海通道的关键节点上设卡，矿要运出去得过这些节点，过就得交钱。

4.6.1 商业模式的三个层次

层次业务内容2025年报毛利率利润特征
第一层：陆港节点费（最核心）口岸清关、保税仓储、货物换装——物理上绕不开的刚性收费59.48%高毛利、相对刚性、护城河最深
第二层：集疏运短腿运费矿区到陆港/铁路装车点的短途公路运输26.73%可竞争、易被铁路挤压、中毛利
第三层：供应链贸易差价焦煤/铜钴矿采购+销售一体化贸易10.47%低毛利、规模大、价格波动敏感

关键洞察：利润金字塔倒挂——毛利最高的业务（陆港59.48%）恰恰是壁垒最深、最不可替代的；毛利最低的业务（贸易10.47%）反而是规模最大但最容易波动的。嘉友的价值锚点在陆港节点，不在运输和贸易。

4.6.2 从单点到网络：当前的局限与未来的可能

嘉友现在处于从单点项目向网络化布局过渡的阶段。卡萨、萨卡尼亚这些单个项目已经能收过路费了，但要真正做到"控制全链路成本、拥有定价权"，得等整个网络建起来。

管理层自己曾表示：在公司没有建立起完善网络、无法通过运营稳定掌控市场价格的时候，对合同物流签订和客户选择非常谨慎。定价权是未来网络成型后才可能有的能力，现在还谈不上。

⚠️ 风险提示：不能简单类比高速公路收费站。收费站修好了就能被动收钱，且无法绕过。但嘉友的公路段理论上可以被铁路绕开（如果洛比托铁路修出支线直接接矿），且特许经营有30年期限，需持续运营维护，面临刚果金和赞比亚政治更迭风险。

4.6.3 三条增长飞轮

飞轮驱动机制当前状态
飞轮一：通车量爬坡公路投产→车流增长→通行费收入爬坡→IRR兑现。卡萨2024年日通车超700车、陆港贡献约2.8亿毛利，萨卡尼亚首段2026年2月刚投产，爬坡数据待观察萨卡尼亚刚启动，验证中

飞轮三：结构升级陆港（毛利率59.48%）占比提升→整体毛利率上行→收入降但利润涨。2026Q1已出现此信号（营收-15.78%、净利+25.18%）已验证！

4.7.1.3 IXM "洛铜出海"的战略意义

IXM不是一家航运公司——它是"非洲铜矿的全球销售总代理"。洛阳钼业2019年以5.18亿美元收购，总部日内瓦，全球第三大基本金属贸易商（仅次于嘉能可、托克），2025年实物贸易量417万吨。

地图上紫色虚线实际代表两条不连通的航线：

  • 西线（大西洋方向）：铜从矿区→洛比托港装船→大西洋→运往欧洲/美洲。对应洛比托走廊的物流路径
  • 东线（印度洋方向）：铜从矿区→达累斯萨拉姆港/德班港装船→印度洋→运往亚洲（中国、东南亚）。对应达累斯萨拉姆通道的物流路径
  • 地图将它们画成一条紫色线仅为示意——表示"无论走西线还是东线，铜最终都进入IXM的全球分销网络，由IXM统一对接宁德时代、格林美等客户"。大西洋和印度洋之间物理上需绕行好望角，不存在横穿非洲大陆的陆路连接

从地图看铜的完整价值链（矿口→全球客户）：

  1. 矿区开采（科卢韦齐 TFM/KFM） →
  2. 公路运输（嘉友卡萨公路·收过路费） →
  3. 口岸通关（嘉友卡萨陆港/萨卡尼亚·收节点费·毛利率59.48%） →
  4. 铁路/公路运输（洛比托铁路或达累斯萨拉姆通道·他人运营） →
  5. 港口装船（洛比托港↗西线/达累斯萨拉姆港↘东线） →
  6. 远洋海运 → IXM交易平台 → 全球客户（宁德时代/格林美等）（洛阳钼业赚贸易利润）

关键洞察：嘉友卡在价值链②③的位置，前有矿区扩产保障货源，后有IXM全球网络保障销路。嘉友是"矿区→港口"的刚需管道——管道收费能力取决于货源和销路两端都畅通。

4.9.4 对嘉友物流的影响分析——"收费公路"模式天然隔离政策风险

关键区分：嘉友是"收费公路运营商"，不是"卡车运输公司"。ARSP 2026年4月的运输本土化强制令要求矿产品公路运输合同50%分配给刚果金本土运输商——这影响的是谁开卡车运货，不影响谁拥有道路收过路费。嘉友按车收费，不管卡车属于谁，过路费照收。

4.9.5 为什么本地加工反而利好嘉友？

禁原矿出口 ≠ 禁矿物流。刚果金的政策目标是"矿石在本地加工后再出口"，不是"不出口"。本地加工需要：

  • 运入：硫酸、石灰石、设备、燃料——这些都需要卡车运进矿区（嘉友双向收费）
  • 运出：粗铜/阴极铜/钴中间品替代原矿出口——产品形态变了，但物流量不降反升
  • 加工厂建设：建筑材料和设备的运输需求

数据验证：Kamoa冶炼厂投产后，硫酸、石灰石等大宗原料进口需求大幅攀升，直接带动卡萨公路日均通车量上升。2026Q1嘉友净利+25.18%（营收-15.78%）的结构优化，部分即源于非洲陆港业务的高毛利增长——而陆港增长又部分受益于加工厂带动的原材料进口物流。

4.9.6 近端风险：运输本土化强制令的执行力度

⚠️ 需要跟踪的变量：

  • 执行时间线：ARSP给出60天合规期，但刚果金矿业商会已公开质疑"从5%到50%在60天内不现实"，最终执行可能延后或分期推进
  • 嘉友的应对选项：①与本土运输商成立合资车队（既满足50%规则又保持运营控制）；②利用萨卡尼亚/莫坎博口岸作为"备用通道"的稀缺性（桥梁坍塌事件已验证嘉友资产的不可替代性）；③嘉友在刚果金注册有本地子公司（JASWIN/SGPCKS），符合"本土企业"资质
  • 历史验证：2026年中期，卡松巴莱萨口岸南部桥梁因洪水坍塌，官方紧急建议车辆绕道嘉友萨卡尼亚口岸——拥堵和路网脆弱性反而凸显了嘉友资产的战略价值

第五章 双轮驱动敏感性分析

嘉友的双变量暴露：焦煤价格（中蒙）与铜矿物流量（非洲）是两个独立变量，单一焦煤价格模拟已无法反映公司盈利结构。

5.1 蒙古焦煤价格敏感性（贸易板块）

⚠️ 测算前提：

  • 基准情景（2025年报口径）：供应链贸易收入51.74亿元，毛利5.42亿元，毛利率10.47%。焦煤均价980元/吨，贸易量约700万吨
  • 关键假设：蒙古焦煤价格变动仅影响贸易板块毛利，跨境物流和陆港业务不受焦煤价格直接影响（分属不同客户群和商品类）
