# 数据归档索引 (data/)

> 数据表达规范见 `docs/DATA_STYLE_GUIDE.md`。

| 目录 | 内容 |

## 数据准确性原则
- 以原始财报 PDF 为优先来源，标注页码；
- 三态标记：`✅ 已核验` / `⚠️ 估算` / `🚫 未披露`；
- 跨报告同口径数据须一致，修订时同步更新源数据与报告。

## 年报/半年报批量下载
- 脚本：`tools/download_reports.py`（仅标准库，需在能访问巨潮/披露易的网络运行）。
- 覆盖 11 家深度研究公司（见脚本 `COMPANIES`）。
- 港股（披露易）：`tools/download_hkex.py` 基于 Playwright 真实浏览器，**已验证可在沙箱直接运行**（关键坑：披露易 `to` 不能超今天、结果需 LOAD MORE 翻页、紫金英文版年报为占位桩需切中文版）。
- 巨潮 A 股：`tools/download_reports.py`（标准库），建议在能直连巨潮的本机终端执行。
